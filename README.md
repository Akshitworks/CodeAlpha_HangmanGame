# 🎯 Hangman Game

A simple and interactive **text-based Hangman Game** built with Python.

The player must guess a randomly selected word one letter at a time before running out of attempts.

---

## 📌 Project Overview

This project was developed as part of a Python programming assignment to demonstrate fundamental programming concepts such as:

- Variables
- Conditional Statements
- Loops
- Lists
- Strings
- Functions
- Classes & Objects
- Input Validation
- Random Module

---

## 🚀 Features

✅ Random word selection

✅ Input validation

✅ Tracks guessed letters

✅ Maximum 6 incorrect attempts

✅ Win/Loss detection

✅ Object-Oriented Programming (OOP)

✅ Clean and beginner-friendly code

---

## 🛠 Technologies Used

- Python 3.x
- Built-in `random` module

No third-party libraries required.
